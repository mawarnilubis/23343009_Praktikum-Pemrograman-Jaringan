## Deskripsi JobSheet

JobSheet 2 membahas **Node.js Module System**, yaitu mekanisme pemecahan program
menjadi beberapa modul agar kode lebih **terstruktur, modular, dan mudah dikelola**.
Pada JobSheet ini juga diterapkan penggunaan **File System (`fs`)** untuk membaca dan
menulis file dalam format **JSON** dan **TXT**, serta pemanfaatan **library pihak ketiga**
seperti `yargs` dan `chalk`.

---

## Tujuan Pembelajaran

Setelah menyelesaikan JobSheet ini, mahasiswa diharapkan mampu:

- Memahami konsep **module system** pada Node.js
- Menggunakan `require()` dan `module.exports`
- Mengelola data menggunakan file **JSON**
- Membuat file teks (**catatan.txt**) menggunakan module `fs`
- Menggunakan library eksternal (`yargs`, `chalk`)
- Membuat aplikasi CLI sederhana berbasis Node.js

---

## Konsep yang Dipelajari

- Node.js Module System
- Core Module `fs`
- File handling (read & write)
- JSON parsing dan stringifying
- Command Line Interface (CLI) dengan `yargs`
- Pewarnaan output terminal dengan `chalk`
