## Deskripsi JobSheet

JobSheet 1 merupakan pengantar dasar penggunaan **Node.js** dalam mata kuliah  
**Praktikum Pemrograman Jaringan**.  
Pada JobSheet ini, mahasiswa mempelajari konsep dasar Node.js sebagai *runtime environment* JavaScript di sisi server serta memahami cara membuat **HTTP Server sederhana** menggunakan *core module* bawaan Node.js tanpa framework tambahan.

---

## Tujuan Pembelajaran

Setelah menyelesaikan JobSheet ini, mahasiswa diharapkan mampu:

- Memahami konsep dasar Node.js
- Menjalankan program JavaScript menggunakan Node.js
- Menggunakan *core module* `http`
- Membuat HTTP Server sederhana
- Menampilkan respon teks melalui browser

---

## Konsep yang Dipelajari

- Node.js sebagai JavaScript Runtime
- Core Module `http`
- Pembuatan server dengan `createServer()`
- Penentuan hostname dan port
- Pengiriman response ke client
